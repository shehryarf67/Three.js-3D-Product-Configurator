# 🎥 3D Product Configurator — Instax Mini 12

A modern, interactive **3D product showcase experience** built with React and Three.js.  
This project recreates a premium, Apple-style product page where users can explore a camera model in real-time, interact with components, and experience smooth scroll-driven storytelling.

---

## ✨ Features

- 🧊 Real-time 3D Model Rendering  
  Built with React Three Fiber (R3F) with smooth orbit controls

- 🎛️ Product Customization  
  Dynamic color changes and component interaction

- 🎬 Scroll-Based Storytelling  
  Smooth section reveals and scroll-driven transitions

- 🎨 Premium UI/UX  
  Clean, Apple-inspired layout with subtle animations

- ⚡ Performance Optimized  
  Efficient rendering using Drei and optimized assets

---

## 🛠️ Tech Stack

- React (Vite)
- Three.js
- React Three Fiber (R3F)
- @react-three/drei
- Leva
- GSAP
- Postprocessing
- CSS

---

## 🚀 Getting Started

Clone the repository:

git clone https://github.com/shehryarf67/Three.js-3D-Product-Configurator.git  
cd Three.js-3D-Product-Configurator  

Install dependencies:

npm install  

Run the development server:

npm run dev
